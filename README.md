     Project Overview

This project focuses on developing a real-time environmental monitoring and control system for beehives.

Bees are very sensitive to changes in temperature and humidity inside the hive. If these conditions become unfavorable, it can affect honey production and even harm the bee colony.

This system monitors the hive environment and automatically controls conditions to keep them suitable for the bees.

The system uses sensors, microcontrollers, and actuators to detect environmental conditions and take necessary actions such as activating a fan to regulate temperature and humidity.

 SYSTEM OPERATION
   Sensors

DHT22 Sensor

Measures temperature

Measures humidity

Sends digital data to the microcontroller.

   Microcontrollers

PIC18F4520

Main controller for processing sensor signals.

Controls actuators such as fans.

ESP32

Used for communication and real-time monitoring.

Can transmit hive data wirelessly.

   Signal Conditioning

LM324 Operational Amplifier

Works as a voltage comparator

Helps process signals from sensors before sending them to the microcontroller.

Cooling System

Fan

Reduces high temperature

Helps reduce high humidity by improving air circulation.

 Valve/motor

Controls airflow or environmental regulation inside the hive.

 Crystal Oscillator

Provides clock timing for the microcontroller to operate correctly.

   Capacitors

Stabilize the circuit.

Reduce noise in the electronic system.

     System Working Principle

 The DHT22 sensor measures temperature and humidity inside the beehive.

 The sensor sends data to the PIC18F4520 microcontroller.

 The microcontroller analyzes the environmental conditions.

 If temperature or humidity is too high, the system activates the fan.

 The fan circulates air inside the hive to stabilize the environment.

 The ESP32 module can transmit the environmental data for monitoring.
